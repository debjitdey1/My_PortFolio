# Portfolio Website
## Description

*This is a personal portfolio website created with HTML, CSS, and JavaScript. It showcases my skills, projects, and a brief introduction about myself as a web developer.*

## Features:

**Responsive Design:** Optimized for both desktop and mobile devices.

**Contact Form:** Integrated a contact form powered by Web3Forms to reach out to me directly.

**Skills Section:** Displays a list of technologies and tools I work with.

**Project Showcase:** A section to display the projects I have built with links to the live sites and source code.

**About Me:** A short introduction of myself and my journey as a web developer.

# Technologies Used

HTML

CSS

JavaScript

Web3Forms (for contact form submission)

# Setup

To view the project locally, follow these steps:

**1.** Clone the repository:

```git clone [https://github.com/debjitdey1/My_PortFolio]```


**2.** Navigate into the project folder:

```cd portfolio```


Open the ```index.html``` file in your browser to view the website.
